# Questionnaire — Professional services

For expertise-led, client-engagement businesses (law, accounting, consulting,
financial, advisory). Ask the shared core (`_router.md`) first, then these.

## Practice

1. What **services / practice areas** do you offer? Which are the priority?
2. Who is the **ideal client** (individuals, SMEs, enterprise; sector; situation)?
3. What **outcome** do clients get from working with you?
4. What's your **approach / method** that sets you apart?

## Credibility (the whole sell)

5. **Credentials**: qualifications, regulatory bodies, memberships, bar/charter.
6. **Track record**: years, clients served, deals/cases, results (anonymised if needed).
7. **Team** — do you want to feature key people, bios, and photos?
8. **Testimonials / case results** you can publish (within confidentiality limits)?
9. Any **publications, speaking, press, awards** that build authority?

## Conversion

10. Primary action — **book a consultation, request a call, send an enquiry**?
11. Is the first consultation **free / paid / fixed-fee**? State it?
12. How do you handle **fees** publicly (fixed, hourly, "request a quote")?
13. Common **questions/concerns** prospective clients raise? → FAQ.

## Tone

14. How **formal** should this feel? Authoritative, approachable, or both?
15. Any **compliance/regulatory** wording that must (or must not) appear?
